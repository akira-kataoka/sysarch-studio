# SysArch Studio — システム構成図エディタ

ブラウザだけで動く、モダンなシステム構成図（アーキテクチャ図）エディタです。
ドラッグ&ドロップで部品を並べ、ノード同士を線でつなぎ、**SVG / PNG** で書き出してスライドや資料にそのまま貼り付けられます。

- 🎛 **依存ゼロ・完全クライアントサイド** — GitHub Pages にそのまま置くだけで動作
- 🖼 **SVG（ベクター・再編集可）/ PNG（2x・4x）書き出し**、PNG クリップボードコピー対応
- 🌌 **Three.js のアンビエント背景** + Obsidian 風グラス UI（明暗テーマ）
- 🧩 **34 種の部品カタログ**（クライアント / ネットワーク / コンピュート / データ / 連携 / セキュリティ運用 / 汎用）
- 💾 JSON で保存・読み込み、ローカル自動保存、Undo/Redo、ミニマップ

## 使い方

| 操作 | やり方 |
|------|--------|
| 部品を置く | 左パレットから**ドラッグ**、またはクリックで中央に配置 |
| つなぐ | ノードにマウスを乗せると出る**端の○をドラッグ**して相手ノードへ |
| 選択・編集 | ノード / 線をクリック → 右のインスペクタでラベル・色・線種などを編集 |
| 移動 | ノードをドラッグ（8px グリッドにスナップ） |
| 画面移動 | 空白をドラッグ、または **Space + ドラッグ** / 中ボタンドラッグ |
| ズーム | マウスホイール（カーソル基準） |
| 全体表示 | ツールバーの倍率表示、または **F** キー |

### ショートカット
`Del` 削除 / `Ctrl+D` 複製 / `Ctrl+Z` 元に戻す / `Ctrl+Shift+Z`（`Ctrl+Y`）やり直し / `Ctrl+S` JSON保存 / `F` 全体表示 / `Esc` 選択解除

### 書き出し
右上の **書き出し ▾** から:
- **SVG** — ベクター。Illustrator / PowerPoint / Figma で再編集可。拡大しても劣化しない
- **PNG (2x / 4x)** — スライド貼り付け向けのラスター画像。背景色付き
- **PNG コピー** — クリップボードへ直接コピー（対応ブラウザ）

> グリッドの点は書き出しには含まれません。図の要素だけがクリーンに出力されます。

## ローカルで動かす

ES Modules を使うため `file://` 直開きではなく簡易サーバー経由で開いてください。

```bash
# どれか一つ
python -m http.server 8000
npx serve .
```

ブラウザで `http://localhost:8000` を開きます。

## GitHub Pages へ公開する

1. このフォルダを GitHub リポジトリとして push
2. リポジトリの **Settings → Pages → Build and deployment → Source** を **GitHub Actions** に設定
3. `main`（または `master`）に push すると `.github/workflows/pages.yml` が自動デプロイ

> `.nojekyll` を含めているので Jekyll による加工は行われません。ビルド不要でそのまま配信されます。

## 構成

```
index.html          画面レイアウト
css/style.css       グラス UI / 明暗テーマ
js/
  main.js           全体の配線（パレット・ツールバー・インスペクタ・デモ・ミニマップ）
  editor.js         SVG 編集エンジン（配置 / 接続 / 選択 / パン&ズーム / Undo / 永続化）
  export.js         自己完結 SVG 生成 + PNG ラスタライズ + クリップボード
  background.js      Three.js アンビエント背景（未ロード時は静的グラデにフォールバック）
  nodes.js          部品カタログ（種別・色・グループ）
  icons.js          ラインアイコン（Feather 風）
```

### 設計メモ
- ノード / 線は **純粋な SVG**（`foreignObject` 不使用）で描画。テーマ変数は描画時に**実際の色値へ解決**して属性に入れているため、書き出した単体 SVG がそのままの見た目で開き、PNG ラスタライズでも canvas が汚染されません。
- Three.js は CDN（unpkg, importmap）から読み込み。ブロックされても背景が消えるだけで編集機能はそのまま動きます。
