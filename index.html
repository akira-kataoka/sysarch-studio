配置

  • 左パレットからドラッグ、またはクリックで中央に追加
  • 「＋ 自前ロゴ」で画像(SVG/PNG)をアップロードしてタイル化
  • 部品名は上の検索で絞り込み。グループ見出しで折りたたみ

接続・配線

  • ノードに乗せて出る端の○をドラッグして相手ノードへ
  • 線を選択 → □(端点)をドラッグで繋ぎ替え
  • 線上の＋点をドラッグで経由点(曲げ)を追加、点をダブルクリックで削除
  • インスペクタで 向き / 線種 / 直角・なめらか を切替

選択・整列

  • クリックで選択、Shift+クリックで複数、Shift+ドラッグで範囲選択
  • 複数選択で整列 / 等間隔分布（右パネル）
  • ドラッグ時に整列ガイドが出て他ノードへ吸着

変形・ゾーン

  • 選択ノードの四隅ハンドルでリサイズ / 矢印キーで微調整(Shiftで1px)
  • ゾーン枠はヘッダーをドラッグで中身ごと移動
  • 「中身に合わせる」「グリッド整列」で枠を自動整頓

画面・タッチ

  • 空白ドラッグ / Space+ドラッグでパン、ホイールでズーム
  • スマホ: 1本指で移動、2本指ピンチでズーム、下部FABで部品/編集
  • 右クリック / ロングタップで複製・前面背面・削除・整列メニュー

書き出し

  • SVG(再編集可) / PNG(2x・4x) / PDF / PNGコピー
  • 「背景色を含める」OFFで透過PNG・SVG
  • JSONで保存・読込、作業は自動保存
